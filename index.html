Python Entry Point

In Python, the entry point refers to the first part of your code that the Python interpreter will execute.

By convention, this is usually managed with a special if __name__ == "__main__": block. 

                
                    def main():
                        print("Hello, World!")

                    if __name__ == "__main__":
                        main()

When you run this script directly, Python will execute main(), printing "Hello, World!".

If you import this script as a module in another script, the main() function won't run automatically because the if condition won't be met.

Printing to the Console

In Python, printing to the console is simple. You can use the print() function to display text, numbers, or even complex data structures.

To print text or numbers, just pass them as arguments to print()

                
                    print("Hello, World!")   # Prints: Hello, World!
                    print(42)                # Prints: 42
                    print(3.14)              # Prints: 3.14

Using print() effectively helps display feedback or debug information, making it an essential tool in any Python project!

                
                    name = "Alice"
                    age = 25
                    print("Name:", name, "Age:", age)   # Prints: Name: Alice Age: 25

You can print multiple items by separating them with commas. print() will automatically add a space between each item

Object Oriented Programming

Python fully supports object-oriented programming (OOP), which lets you create classes, objects, and methods to organize code into reusable components

Here is an example:

                
                    class Animal:
                        def __init__(self, name):
                            self.name = name

                        def speak(self):
                            raise NotImplementedError("Subclasses must implement this method")

                    class Dog(Animal):
                        def speak(self):
                            return "Woof!"

                    class Cat(Animal):
                         def speak(self):
                             return "Meow!"

                    animals = [Dog("Rover"), Cat("Whiskers")]
                    for animal in animals:
                        print(f"{animal.name} says {animal.speak()}")
